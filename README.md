# guia-modelo-osi# 🌐 Guia Prático: O Modelo OSI

Este repositório foi criado para documentar meus estudos sobre Redes de Computadores, focando no **Modelo OSI (Open Systems Interconnection)**, um padrão de referência fundamental para a comunicação entre sistemas.

## 📚 As 7 Camadas do Modelo OSI

O modelo é dividido em 7 camadas, lidas da camada mais alta (onde o usuário interage) até a mais baixa (meio físico).

* **7. Aplicação (Application):** É a interface direta com o usuário e os aplicativos (ex: HTTP, FTP, SMTP).
* **6. Apresentação (Presentation):** Formata, traduz e criptografa os dados para que a camada de aplicação possa entendê-los (ex: SSL, TLS, JPEG).
* **5. Sessão (Session):** Estabelece, gerencia e encerra as conexões (sessões) entre os computadores.
* **4. Transporte (Transport):** Garante a entrega dos dados de ponta a ponta sem erros, gerenciando o fluxo e a correção de falhas (ex: TCP, UDP).
* **3. Rede (Network):** Responsável pelo roteamento dos pacotes de dados através das melhores rotas possíveis (ex: IP, ICMP).
* **2. Enlace de Dados (Data Link):** Controla o acesso ao meio físico e detecta erros na transmissão local (ex: Ethernet, MAC Address).
* **1. Física (Physical):** É a transmissão bruta dos bits (0s e 1s) através do cabo, fibra ótica ou sinal de rádio.
